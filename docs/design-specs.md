# 🎨 设计规范文档

## 设计原则

- **简洁至上**：最少点击次数完成核心任务
- **macOS 原生感**：遵循 Apple Human Interface Guidelines
- **渐进式披露**：默认简单，高级选项可展开
- **即时反馈**：所有操作有明确的状态反馈

## 色彩系统

| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色 | 蓝色 | #007AFF (macOS accent) |
| 成功 | 绿色 | #34C759 |
| 警告 | 橙色 | #FF9500 |
| 错误 | 红色 | #FF3B30 |
| 背景 | 浅灰 | #F5F5F7 |
| 文字 | 深灰 | #1D1D1F |
| 辅助文字 | 中灰 | #86868B |

## 交互规范

| 场景 | 行为 |
|------|------|
| 拖拽文件进入 | 拖放区域高亮，显示绿色边框 |
| 转换中 | 主按钮显示进度环，禁用参数调节 |
| 转换成功 | 短暂显示 ✅ 动画，文件自动保存 |
| 转换失败 | 显示具体错误信息，提供重试按钮 |
| 批量处理 | 每个文件独立状态，失败不影响后续 |

---

## 图标与资源

- 应用图标：基于 SF Symbols 风格（macOS）
- 文件格式：SVG/PNG 图标资源
- 空状态插图：简洁线条风格

---

## 无障碍

- 支持完整键盘导航 (Tab/Enter/Esc)
- 所有控件有 accessible name/label
- 颜色不是唯一的信息传递方式
- 支持系统字体大小缩放
