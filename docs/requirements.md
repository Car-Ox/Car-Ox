# 📋 开发需求文档

## 项目概述

**项目名称：** LiveToGif

**项目描述：** 将 Apple Live Photo 转换为高质量 GIF 动图的跨平台桌面应用

**目标平台：** macOS（主）/ Windows（辅）

**目标用户：** 需要将 Live Photo 分享到不支持 Live Photo 平台的用户

---

## 功能需求

### FR-01: Live Photo 导入
- 支持拖拽导入 .MOV + .HEIC 文件对
- 支持批量导入
- 支持从相册直接导入（macOS）

### FR-02: GIF 转换
- 将 Live Photo 视频片段转换为 GIF
- 可调节帧率 (5-30 FPS)
- 可调节输出尺寸
- 可选颜色优化/抖动算法

### FR-03: 预览功能
- 实时预览转换效果
- 逐帧浏览
- 原始与转换后对比

### FR-04: 导出功能
- 导出为 GIF 格式
- 自定义导出路径
- 保留元数据选项

### FR-05: 批量处理
- 多文件队列转换
- 后台处理
- 进度显示

---

## 非功能需求

| 类别 | 要求 |
|------|------|
| 性能 | 单文件转换 < 5 秒 |
| 可靠性 | 批量处理不崩溃，错误隔离 |
| 可用性 | 直观的拖拽界面，3 次点击内完成转换 |
| 兼容性 | macOS 12+, Windows 10+ |
| 可维护性 | 模块化架构，单元测试覆盖率 > 80% |

---

## 用户故事

1. **作为** iPhone 用户，**我想要** 将 Live Photo 转为 GIF，**以便** 在社交媒体上分享
2. **作为** 设计师，**我想要** 批量转换多个 Live Photo，**以便** 高效准备素材
3. **作为** 普通用户，**我想要** 拖拽文件即可转换，**以便** 无需学习复杂操作
